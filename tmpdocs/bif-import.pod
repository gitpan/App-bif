=head1 NAME

bif-import -  import projects from a remote hub

=head1 VERSION

0.1.0_10 (2014-04-17)

=head1 SYNOPSIS

    bif import PATH... HUB

=head1 DESCRIPTION

Import projects from a hub.

=head1 ARGUMENTS

=over

=item PATHS...

The path of the remote project to import.

=item HUB

The alias of a previously registered hub.

=back
